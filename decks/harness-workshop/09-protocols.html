MCP & SubAgents

MCP — Model Context Protocol

Protocols are now load-bearing. MCP servers provide tools dynamically — the harness connects on startup, discovers available tools, and makes them available to the agent.

// mcp.json
{
  "playwright": {
    "command": "npx",
    "args": ["@playwright/mcp"]
  },
  "github": {
    "command": "npx",
    "args": ["@github/mcp-server"]
  }
}
