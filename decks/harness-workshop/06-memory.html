Memory

Context is precious. Context Rot is real.

As the context window fills up, model performance degrades — reasoning gets worse, instructions get lost, costs spike. This is Context Rot. The harness needs strategies to fight it.

🗜️

Compaction

When context fills up, intelligently summarize and compress — don't just truncate. The agent keeps working with a clean window.

📤

Tool Call Offloading

Large tool outputs clutter context without adding value. Keep head + tail, offload the rest to the filesystem for on-demand access.

💰

Prompt Caching

Stable prompt prefix = cache hits. Compaction preserves the prefix so caches stay warm and costs stay low.
