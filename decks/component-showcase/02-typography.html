BODY TEXT

Body Text

Paragraph and inline styles for readable, scannable content.

This is a regular paragraph styled at 1.15rem with a 1.7 line-height. It uses var(--text-secondary) for comfortable reading on dark backgrounds. The max-width is capped at 680px to maintain a readable measure.

p — 1.15rem / line-height 1.7

Use strong tags for bold emphasis — rendered at 600 weight in var(--text-primary).

<strong> — font-weight 600, color var(--text-primary)

Inline code snippets use the monospace font stack defined by --font-mono.

<code> — font-family var(--font-mono)

The .mono class also applies var(--font-mono) to any element.

.mono — font-family var(--font-mono)
